So sánh 2 file văn bản để hỗ trợ dịch thuật

Công dụng:

Chương trình này giúp phân tích 2 file văn bản để:
  - "align": so sánh và tách nội dung 2 văn bản thành từng cặp câu. (ví dụ: so sánh tài liệu gốc và tài liệu dịch ở 2 ngôn ngữ)
  - "compare": so sánh sự tương đồng (trong từng câu) giữa 2 phiên bản khác nhau của một tài liệu. (ví dụ: để tìm sự khác biệt giữa 2 lần xuất bản)
Chương trình sẽ xuất ra file ZIP chứa các file XLS gồm: (cột A) câu ở file 1, (cột B) câu ở file 2, (cột C) chỉ số tương đồng giữa 2 câu.

Gợi ý: dùng chế độ conditional formatting trong Excel 2010 trở về sau để tô màu cho cột C tương ứng với giá trị tương đồng:
  Xem hướng dẫn để vẽ "data bar" (một bản dịch tiếng Việt) cho cột C, với giá trị từ 0 đến 1. Sau đó lưu file ở dạng .XLSX. (trong LibreOffice Calc cũng có chức năng "conditional formatting").
  Các ô biểu thị sự giống nhau giữa từng cặp câu sẽ có dạng: https://www.excel-easy.com/examples/images/data-bars/data-bars.png

Hướng dẫn:

  + Chọn 2 file văn bản để upload. 2 file này phải cùng kiểu (cùng có tên mở rộng là .DOC, .DOCX, .TXT, hoặc .PDF).
  + Chọn ngôn ngữ trong các file văn bản. (mặc định là de - tiếng Đức)
  + Nếu 2 file là PDF: chương trình sẽ so sánh từng trang. Hãy điền vào chuỗi trang để so sánh. (mặc định là trang 1)
  * Lưu ý: nếu so sánh cùng lúc quá nhiều trang, sẽ quá thời gian cho phép xử lý của trang web. Chuỗi xử lý chỉ nên có tối đa 15 trang.
  Các chuỗi trang trong file 1 và trong file 2 phải có cùng số trang.

IMPORTANT: Bạn có muốn XÓA file tải lên sau khi nó được xử lý xong? (kết quả đầu ra sẽ vẫn được lưu trên trang web này)
(default) Có, xóa dữ liệu tôi tải lên sau khi đã xử lý xong Không, dữ liệu tải lên sẽ được lưu lại ở trang web để phục vụ việc debug
